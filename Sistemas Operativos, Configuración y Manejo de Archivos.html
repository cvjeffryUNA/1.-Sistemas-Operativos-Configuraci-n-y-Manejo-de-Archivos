¿Qué es un Sistema Operativo?

Definición

Un Sistema Operativo (SO) es un conjunto de programas que actúan como intermediario entre el usuario y el hardware del computador.

Objetivo Principal

Gestionar los recursos físicos y lógicos del sistema y proporcionar una interfaz de uso segura y eficiente.

Ejemplos

Funciones Principales

Administración de Procesos

  • • Controla ejecución de programas
  • • Asigna tiempo de procesador
  • • Permite multitarea

Gestión de Memoria

  • • Asigna memoria RAM
  • • Libera memoria no usada
  • • Administra memoria virtual

Gestión de Dispositivos

  • • Controla periféricos
  • • Usa controladores (drivers)
  • • Reconoce hardware

Gestión de Archivos

  • • Organiza datos
  • • Sistemas: NTFS, ext4, APFS
  • • Protege información

Interfaz de Usuario

  • • GUI: ventanas, iconos
  • • CLI: línea de comandos
  • • Interacción usuario-sistema

Seguridad

  • • Control de acceso
  • • Protección contra malware
  • • Respaldo y recuperación
